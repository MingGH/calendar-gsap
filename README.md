## 🌌 忍者日历 · Cyber Animal Calendar

![](https://img.996.ninja/ninjutsu/085e14d1d5fc10796970ac64f0365cfd.png)

只需在浏览器中打开 index.html 文件即可使用。

```shell
# 克隆项目
git clone <repository-url>

# 进入项目目录
cd cyber-animal-calendar

# 打开 index.html
open index.html
```
